    • file structure

    choose a version to start by folder name below

    • documentation - help document
    • static - static header version
    • slideshow - slideshow header version
    • video - youtube video header version

    each version folder have below file

    • assets/audio - audio file (not included)
    • assets/css - style file
    • assets/fonts - font
    • assets/img - image (demo image not included)
    • assets/js - javascript file
    • assets/php - working contact form and subscribe form
    • index.html - main html file

    you can switch to other version at anytime by change the variable, so choose any version to start to just same, but we have pre config the js file.

basic

preloader

open assets/js/variable.js find

            var __preloaderFadeOut = 1200; // fadeout
            var __preloaderDelay = 800; // delay

change the preloader time, for preloader image, open assets/css/style.css find

            background: #000; /* preloader background color */

change preloader background color

and find

            #preloader-img {
              position: absolute;
              top: 50%;
              left: 50%;
              background: url(../img/preloader.gif) no-repeat 50% 50%; /* preloader image */
              width: 60px; /* preloader image width */
              height: 60px; /* preloader image height */
              margin-top: -30px; /* half of preloader image height */
              margin-left: -30px; /* half of preloader image width */
            }

preloader image place at assets/img/preloader.gif, follow above comment to change the preloader image size, e.g. if your preloader image is 100 width, 260 height, then change css like below

              background: url("../img/preloader.gif") no-repeat center; /* preloader image */
              width: 100px; /* preloader image width */
              height: 260px; /* preloader image height */
              margin-top: -50px; /* half of preloader image height */
              margin-left: -130px; /* half of preloader image width */
            }

disable section

if need to disable a section

open assets/js/variable.js find

            var __disableSection = {
              service : false, // service
              about : false, // about
              philosophy : false, // philosophy
              subscribe : false, // subscribe
              contact1 : false, // contact form
              contact2 : false // contact info
            };

for example, we need disable subscribe change to

            var __disableSection = {
              service : false, // service
              about : false, // about
              philosophy : false, // philosophy
              subscribe : true, // subscribe
              contact1 : false, // contact form
              contact2 : false // contact info
            };

very easy, change false to true, it will remove section and nav icon

slideshow header

open assets/js/variable.js find

            var __headerStyle = 2; // 1 = static image, 2 = slideshow, 3 = youtube video

change variable to 2

find

            var __imageAmount = 3; // image amount
            var __slideshowDuration = 5000; // duration
            var __slideshowDelay = 800; // delay

e.g. we want 5 slideshow images, 3 seconds duration time and 1.2 second delay time, change to

            var __imageAmount = 5; // image amount
            var __slideshowDuration = 3000; // duration
            var __slideshowDelay = 1200; // delay

slideshow image place at assets/img/bg/ name slideshow-01.jpg, slideshow-02.jpg ... slideshow-11.jpg etc

youtube video header

open assets/js/variable.js find

            var __headerStyle = 3; // 1 = static image, 2 = slideshow, 3 = youtube video

change to 3

find

            var __youtubeUrl = 'https://www.youtube.com/watch?v=GKc0TUElTZM'; // youtube video url
            var __videoStart = 77; // start time (seconds)
            var __videoEnd = 122; // end time (seconds)
            var __videoLoop = true; // loop
            var __videoMute = false; // mute on start

change the url to your youtube video link, start time and end time 1 mean 1 seconds, video mute mean mute at video start

video wont active at mobile/table/ie8, place a image at assets/img/bg/video-mobile.jpg for mobile/table, and set a image for ie8 and before video loaded, place at assets/img/bg/video-desktop.jpg

background overlay

to set overlay, open assets/js/variable.js find

            /*=================================================
            overlay
            =================================================*/
            var __overlayStyle = 2; // 1 = normal, 2 = gradient
            var __overlayOpacity = 0.75; // overlay opacity, 0 to 1

1 for normal overlay, 2 for gradient, __overlayOpacity for opacity

to change overlay color, open assets/css/color/color.css, color.css mean the color style file, name as red.css blue.css etc...find

            #BE2121 - gradient color 1
            #2069b0 - gradient color 2
            #19191D - overlay color

find and replace above color code to change, we have place 6 style gradient color inside red blue green and yellow purple css and default style.css, copy it to custom.css if need

audio

open assets/js/variable.js find

            var __audio = on; // audio toggle

set true to enable background music, audio file place at assets/audio/audio.mp3, audio file not included at item

audio only active when youtube video background disable, because html5 reason, mobile cant play audio auto start, will play with a button. ie8 not support this audio function.

header effect

item come with 3 header effect

star

open assets/js/variable.js find

            /*=================================================
            header effect
            =================================================*/
            var __effect = 1; // 0 = disable, 1 = star, 2 = cloud

set the value like above, star effect will enable, ie8/mobile will force to disable

cloud

open assets/js/variable.js find

            /*=================================================
            header effect
            =================================================*/
            var __effect = 2; // 0 = disable, 1 = star, 2 = cloud

set the value like above, cloud effect will enable, ie8/ie9/mobile will force to disable

cloud effect will slowdown browser in some case

subscribe (mailchimp)

  • 1. first you need a mailchimp account, api, and list id, for more info please view http://mailchimp.com/
  • 2. and then find your mailchimp form id, login mailchimp, select your list, click Signforms > Embedded forms > find Copy/paste onto your site
  • 3. now search
            form action=

you will see something like that

            http://xxx.xxx.list-manage.com/subscribe/post?u=xxxxxx&id=xxxxxx

that link is your mailchimp post url, then next step

open assets/js/variable.js find

            /* mailchimp */
            var __mailChimp = false; // true = mailchimp form, false = php subscribe form
            var __mailChimpUrl = 'MAILCHIMP_POST_URL_HERE'; // mailchimp post url

set to true and change to your mailchimp post url
